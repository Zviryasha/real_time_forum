
### Need to check cookies for auth token
Is token not exist :
 - **default** login page `/login` 
 - or register page if register button was pressed `/register`

Token exist :
	  -  **default**  `/home`
	  -  if pressed **new post**  -> `/new-post`
	  -  if pressed **post**  -> `/post`
	  -  if pressed **profile**  -> `/profile`
